--- author: fileformat comments: true date: 2007-05-19 11:07:00+00:00 layout: post slug: tab-size-in-vi title: Tab size in vi wordpress_id: 18 tags: - tabs - tabsize - tabstops - vi - vim --- I use 4 spaces for tabs but the default vi (aka vim) in Debian Etch uses 8 spaces. To make it default to 4, create a .vimrc file in your home directory with the line: 
:set tabstop=4
